# 🚀 EduSphere — Next-Gen Virtual Classroom

**EduSphere** is a modern, browser-based virtual classroom designed to help learners master full-stack web development through interactive, real-time, and project-driven learning.

---

## 🌐 Overview

EduSphere combines structured curriculum, live sessions, built-in development tools, and collaboration features into a single seamless platform.

> "Where learning comes alive."

---

## ✨ Features

### 📚 Deep Curriculum
- Structured learning path:
  - HTML → CSS → JavaScript → TypeScript
- Project-based lessons
- Hands-on learning approach

### 🎥 Live Classroom
- Real-time instructor-led sessions
- Camera & microphone support
- Screen sharing functionality

### 🛠️ Built-in Dev Tools
- Code editor inside browser
- Live preview environment
- Terminal & debugging tools

### 🤝 Collaboration
- Pair programming
- Group projects
- Live chat & code review

### 📖 Learning Resources
- Cheat sheets
- References for HTML, CSS, JS
- Deep-dive explanations

### 🧪 Quizzes & Tracking
- Unit-based quizzes
- Instant feedback
- Progress tracking system
